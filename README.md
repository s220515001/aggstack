# AggStack (Aggregated Stack)

[中文版本](README_zh.md)

---

## What is AggStack?

AggStack is a Fabric mod for Minecraft 1.21.2+ that allows stacking items with different NBT data (enchantments, durability, potion effects, etc.) into a single inventory slot.

No new blocks. No new items. Just a better inventory.

## Features

- **Merge different NBT items** – Swords with different enchantments, tools with different durability, potions with different effects. All can share one slot.
- **Expandable panel** – Press a key (default: Left Alt) to see all sub-items in a grid.
- **Quick switching** – Hold Ctrl + Scroll to cycle through primary items.
- **Blacklist system** – Disable auto-aggregation for specific items or potion effect types.
- **Potion NBT differentiation** – Normal, splash, lingering, and each effect type are separate.
- **Death drop control** – Choose between splitting into individual items or keeping the stack as-is.
- **Configurable** – GUI config screen with keybindings, behavior options, and blacklist management.
- **No overhead** – The mod doesn't add any new blocks, items, or crafting recipes.

## Usage

| Action | How to |
|--------|--------|
| Create an aggregated stack | Drag two similar items with different NBT together |
| View sub-items | Hover + hold expand key (default: Left Alt) |
| Switch primary item | Hold Ctrl + Scroll |
| Extract one sub-item | Shift + Left Click on the aggregated stack |
| Open config screen | Press G (default) |
