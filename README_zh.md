# AggStack（聚合堆叠）


[English Version](README.md)

---

## 这是什么模组？

AggStack 是一个 Fabric 模组，适用于 Minecraft 1.21.2+。它允许将 **NBT 不同** 的物品（附魔、耐久、药水效果等）堆叠到同一个格子里。

不增加任何新方块、新物品。只是让背包更好用。

## 功能

- **合并不同 NBT 的物品** – 不同附魔的剑、不同耐久的工具、不同效果的药水，都能放进一个格子
- **展开面板** – 按下快捷键（默认左 Alt）可查看所有子物品
- **快速切换顶层物品** – 按住 Ctrl + 滚动滚轮
- **黑名单系统** – 可禁用特定物品或某类药水的自动聚合
- **药水 NBT 细分** – 普通、喷溅、滞留型以及不同药水效果各自独立
- **死亡掉落控制** – 可选择拆分成单个物品掉落，或整体保留聚合槽
- **可配置** – 图形化配置界面，支持快捷键、行为选项和黑名单管理
- **无额外负担** – 模组不添加任何新方块、新物品或合成配方

## 使用方法

| 操作       | 方式                    |
|----------|-----------------------|
| 创建聚合槽    | 将两个 NBT 不同的同类物品拖到一起   |
| 查看子物品    | 鼠标悬停 + 按住展开键（默认左 Alt） |
| 切换顶层物品   | 按住 Ctrl + 滚动滚轮        |
| 拆分出一个子物品 | Shift + 左键点击聚合槽       |
| 打开配置界面   | 按 G（默认）               |
